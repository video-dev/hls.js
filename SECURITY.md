# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >= 1.6  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Report vulnerabilities [using this link](https://github.com/video-dev/hls.js/issues/new?assignees=&labels=Bug%2CNeeds+Triage%2CSecurity&template=bug.yaml). Project maintainers usually respond within a week.

Contributions via pull request are welcome. They will be [scheduled with a milestone](https://github.com/orgs/video-dev/projects/4) once accepted.
